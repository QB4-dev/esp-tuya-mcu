# TUYA MCU emulator for ESP-IDF

ESP-IDF component for emulating TUYA MCU modules on ESP8266/ESP32


## Installation

### Using ESP Component Registry (Recommended)

```bash
idf.py add-dependency "QB4-dev/esp-tuya-mcu^1.0.0"
```

### Manual Installation

Clone this repository into your project's `components` directory:

```bash
cd your_project/components
git clone https://github.com/QB4-dev/esp-tuya-mcu.git esp-tuya-mcu
```
